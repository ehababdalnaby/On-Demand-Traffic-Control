On-Demand Traffic Control
System static architecture

system description

  • This system consist of two modes normal mode and pedestrians mode.
  • It controls two traffic lights one for cars and one for pedestrians.
  • It has button if the button pressed the system will change from normal mode to pedestrians mode.

Hardware

  • 6 LEDs (3 for cars and 3 for pedestrians).
  • 1 push button.
  • ATmega32 MCU.

Software & IDEs

  • Proteus (used for system simulation).
  • Microchip studio (used for development and debugging).

system design

  • This system consist of 4 layers (MCAL layer,ECUAL layer,Application layer,Service layer)

State machine

  • This state machine describes the flow of the system
  • It consist of 7 main states
    NORMAL_GREEN
    NORMAL_YELLOW_AFTER_GREEN
    NORMAL_RED
    NORMAL_YELLOW_AFTER_RED
    BUTTON_PRESSED_AT_GREEN
    BUTTON_PRESSED_AT_YELLOW
    BUTTON_PRESSED_AT_RED
