Bit 2:0 - CS02:0: Clock Select
  • The three Clock Select bits select the clock source to be used by the Timer/Counter and located in TCCR0.
    CS02 CS01 CS00 Description
    0 0 0 No clock source
    (Timer/Counter stopped).
    0 0 1 clkI/O/(No prescaling).
    0 1 0 clkI/O/8 (From prescaler).
    0 1 1 clkI/O/64 (From prescaler).
    1 0 0 clkI/O/256 (From prescaler).
    1 0 1 clkI/O/1024 (From prescaler).
    1 1 0 External clock source on T0 pin.
    Clock on falling edge.
    1 1 1 External clock source on T0 pin.
    Clock on rising edge.
    Clock Select Bit Description
    Note
    If external pin modes are used for the Timer/Counter0, transitions on the T0 pin will
    clock the counter even if the pin is configured as an output. This feature allows software
    control of the counting.
