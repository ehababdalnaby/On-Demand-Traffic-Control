EN_pinErro_t DIO_pinInit ( EN_pinNum_t  pinNum,
EN_pinDirection_t  pinDirection 
)

Set the direction of the pin.

DIO_pinInit

  • This function makes pin input or output.
  • it makes the pinNum Output by setting the pinNum in the DDRx (x:A,B,C or D) register.
  • it makes the pinNum Input by clearing the pinNum in the DDRx (x:A,B,C or D) register.
Parameters
[in]pinNumit represent the pin number (PA0 to PD7).
[in]pinDirectionit represent the pin direction it may be (Input or Output).
[out]noneno output arguments
Return values
WRONG_PIN_NUMif the pinNum is wrong.
WRONG_PIN_DIRif the pinDirection is wrong.
OKif the pinNum and the pinDirection are correct.
